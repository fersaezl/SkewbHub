# SkewbHub

A web-based Skewb algorithm trainer to practice cases and track your times.

## Features
- Timer with spacebar control
- Case visualization
- Session statistics (ao5, ao12, pb)
- Case selection by method

## Tech Stack
HTML · CSS · Bootstrap · JavaScript · jQuery
